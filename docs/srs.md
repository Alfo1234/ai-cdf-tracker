1. Introduction
The AI-Powered CDF Tracker is a web platform that processes CDF-related documents (PDFs, reports, project lists), extracts structured data, applies AI analysis (summaries + anomaly detection), and displays results on a public dashboard to improve transparency, accountability, and citizen awareness.

2. System Objectives
Provide clear transparency on how each constituency uses CDF funds.
Detect irregular spending patterns using machine learning.
Offer simplified AI summaries of complex reports.
Enable citizens to easily access project information.
Provide admin users with easy tools to upload and process documents.
Support national-level analytics and a Public Accountability Score.

3. Stakeholders
Citizens
Journalists & civil society
Auditors
NG-CDF administrators
Developers & maintainers

4. High-Level Features
Document upload (PDF, scanned PDF, images)
OCR + text extraction
Automated project classification
AI summarization
Anomaly detection
Public dashboard (map + constituency pages)
Admin panel
Citizen feedback system

5. System Boundaries
Out of scope:
Real-time financial APIs
Full-scale government integration
Authentication for citizens (view-only)
